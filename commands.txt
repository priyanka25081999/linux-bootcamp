Some basic linux commands:

ls    -  List files or directories
cat   -  Type out concentrate the files
touch -  Either create a file or update the timestamps of the file
rm    -  Remove a file
mv    -  Move or rename files
mkdir -  Create a directory
rmdir -  Remove a directory
wc    -  Count lines, words, bytes in a file
ln    -  Create hard/symbolic links
file  -  Show the type of a file
tail  -  Look at end of a file
head  -  Look at beginning of a file
less  -  Displays file content or command output one page at a time in the terminal
more  -  Displays file content or command output one page at a time in the terminal
